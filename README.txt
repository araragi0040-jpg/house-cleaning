CleanFlow v005 マスタ設定版

■ 起動方法
1. ZIPを展開します。
2. index.htmlをダブルクリックしてブラウザで開きます。

■ v005で追加・変更した内容
・管理者メニューに「設定」を追加
・元請けを画面から追加可能
・担当者を画面から追加可能
・元請け別に、各作業の請求単価と担当者支払単価を編集可能
・登録した元請け、担当者、単価を案件登録へ自動反映
・スタッフ画面の担当者切替も登録済み担当者へ自動反映
・案件詳細から既存案件を複製可能
・バックアップ／復元に元請け、担当者、単価設定も含めるよう変更
・v004までの案件データと請求書設定を引き継ぐ仕様

■ おすすめの確認手順
1. 管理者画面の「設定」を開く
2. 元請けまたは担当者を1件追加する
3. 元請けを選び、作業別単価を変更して保存する
4. 「案件追加」を開き、追加した元請け・担当者が選べることを確認する
5. 作業内容を選び、保存した単価が自動入力されることを確認する
6. 案件詳細の「複製」から、同様の案件を少ない入力で登録する

■ データ保存について
・現段階では案件、写真、設定をブラウザ内に保存します。
・複数端末での共有、ログイン、クラウド写真保存は未接続です。
・正式運用へ進める際は、現在の画面構成を維持したままクラウド保存へ切り替えます。
